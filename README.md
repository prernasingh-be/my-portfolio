# Nova — Modern E-commerce Shop

A modern, responsive e-commerce shop built with vanilla HTML, CSS, and JavaScript.

## Features

- **Homepage** with hero banner, featured products, promo banner, and feature highlights
- **Shop page** with filtering by category and sorting (price, name)
- **Dynamic shopping cart** in a slide-in drawer with quantity controls
- **Cart persistence** via `localStorage`
- **Navigation bar** with logo and cart icon showing live item count
- **Footer** with grouped links
- Fully **responsive** design (desktop, tablet, mobile)
- Clean, modern aesthetic using Playfair Display + Inter typography

## Structure

```
├── index.html      # Homepage
├── shop.html       # Product listing page
├── css/style.css   # All styles
└── js/
    ├── products.js # Product catalog
    └── app.js      # Cart logic & rendering
```

## Usage

Open `index.html` in a browser — no build step required.
